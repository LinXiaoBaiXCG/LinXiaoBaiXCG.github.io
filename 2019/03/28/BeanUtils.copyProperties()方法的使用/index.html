BeanUtils.copyProperties()方法的使用

2019-03-28
  • 编程

BeanUtils简介：

一个JavaBean通常包含了大量的属性，很多情况下，对JavaBean的处理导致大量get/set代码堆积，增加了代码长度和阅读代码的难度。BeanUtils 是 Apache commons组件的成员之一，主要用于简化JavaBean封装数据的操作。

BeanUtils.copyProperties()方法源码

Object source指的是来源对象，Object target指的是目标对象。意思是将Object source的属性拷贝到Object target。

实例

将一个JavaBean对象的属性赋值给另一个JavaBean对象

学生信息类

教师信息类

控制台输出：Student(name=我是学生, age=0, sex=null)
Student(name=我是教师, age=20, sex=null)

注意

  1. 不管目标对象（Student）是否有值，相同的属性都会被拷贝。
  2. 当来源对象（Teacher）中有属性值为null时，即使目标对象相同的属性有值，目标对象的属性同样被拷贝，值也为null。
  3. 只要两个JavaBean对象的属性名称名称相同就会赋值，不同的不赋值。
